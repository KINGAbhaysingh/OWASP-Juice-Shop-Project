# OWASP Juice Shop Security Assessment

## Objective
Performed a security assessment on OWASP Juice Shop.

## Tools Used
- Kali Linux
- Burp Suite
- OWASP Juice Shop
- Firefox

## Project Workflow
1. Set up Juice Shop
2. Configure Burp Suite
3. Analyze traffic
4. Document findings
5. Create recommendations
